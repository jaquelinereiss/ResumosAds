Unidade Central de Processamento (CPU)

A Unidade Central de Processamento, também conhecido como CPU, é um dispositivo responsável pelo processamento de dados em um computador, celular e outros dispositivos eletrônicos, funcionando como o “cérebro” desses equipamentos

A Unidade Central de Processamento (CPU) é dividida em três unidades essenciais, sendo elas:

  • UC: Unidade de Controle – circuito que gera sinais elétricos para organizar como e quando as instruções devem ser executadas. Esses sinais direcionam a comunicação entre a CPU e componentes como memória RAM e dispositivos de entrada e saída de dados. A unidade de controle (UC) é um componente da CPU que direciona a operação do processador. Uma UC tipicamente usa um decodificador binário para converter instruções codificadas para sinais de controle temporizados que orquestram a operação de outras unidades (memória, unidade lógico-aritmética, periféricos, etc). Sem a UC, o computador seria uma máquina de função única, não sendo possível o processador funcionar com propósito geral. A maior parte dos recursos de um computador são administrados pela UC. Ela direciona o fluxo de dados entre a CPU e os outros componentes do computador. John von Neumann incluiu a UC como parte da arquitetura de von Neumann.Em designs modernos de computador, a unidade de controle é tipicamente uma parte interna da CPU, com a sua função e operações intocadas desde a sua introdução.
  • ULA: Unidade Lógica e Aritmética – um circuito responsável por executar operações lógicas, cálculos como adição e multiplicação, além de operações booleanas complexas. A unidade lógica e aritmética (ULA), em inglês arithmetic logic unit (ALU), é um circuito digital que realiza operações de adição e booleana AND. A ULA é uma peça fundamental da unidade central de processamento (UCP), e até dos mais simples microprocessadores. É na verdade uma grande calculadora eletrônica.
  • Registradores – tipo de memória rápida e volátil (perdem dados ao ficar sem energia elétrica) usados em pequenas quantidades para armazenamento de dados. A sua função é permitir que a CPU tenham acesso imediato a dados e instruções necessárias para a execução de operações lógicas ou aritméticas. O registrador de uma CPU (unidade central de processamento) é a memória dentro da própria CPU que armazena o número de bits. Os registradores estão no topo da hierarquia de memória, sendo assim, é um tipo de memória mais rápida e financeiramente mais custosa. Apesar do alto custo por bit armazenado, sua velocidade de acesso é essencial para o funcionamento dos computadores modernos e, portanto, são incluídos, ainda que em menor capacidade, mesmo em processadores de baixo custo. Os registradores são circuitos digitais capazes de armazenar e deslocar informações binárias, e são tipicamente usados como um dispositivo de armazenamento temporário. São utilizados na execução de programas de computadores, disponibilizando um local para armazenar dados. Na maioria dos computadores modernos, quando da execução das instruções de um programa, os dados são deslocados da memória principal para todos os registradores. Então, as instruções que utilizam estes dados são executadas pelo processador e, finalmente, os dados são movidos de volta para a memória principal.

Barramentos

os componentes de um computador, como processadores, memórias, placas de vídeo e diversos outros, são conectados à placa-mãe a partir do que chamamos de barramento. Os barramentos são o encaixe que cada peça precisa para funcionar corretamente.

Há três funções distintas nos principais barramentos de um computador, que, em termos simples, conectam o processador, a memória e os outros componentes conectados a ele pelo que chamamos de barramentos de entrada e saída.

  • Barramento de dados – é por este tipo de barramento que ocorre as trocas de dados no computador, tanto enviados quanto recebidos.
  • Barramento de endereços – atua como um regulador das outras funções, podendo limitá-las ou expandi-las em razão de sua demanda.
  • Barramento de controle – Onde acontece o processamento das informações. Exemplo: processador do computador.

Tecnologias empregadas nos processadores

  • CISC – O CISC (Complex Instruction Set Computers) é um sistema com um conjuntos de instruções complexas, mais usada em computadores e notebooks.
  • RISC – O RISC (Reduced Instuction Set Computes) é um sistema com um conjunto de instruções reduzidas, mas comum em smartphones e tabletes.

Memória Principal

Conheça os papéis das memórias de um computador navegando com o mouse em cada uma delas:

Memória Cache
A memória cache é uma memória de acesso rápido que armazena dados frequentemente utilizados para melhorar o desempenho do sistema. Agiliza a comunicação do processador com a memória principal, proporcionando um processamento mais eficiente
Registradores
Os registradores são uma pequena quantidade de memória de alta velocidade localizada na CPU, usada para armazenar dados temporários e instruções de programas em execução. Uma função básica da memória do computador é de armazenas dados que serão processados, os registrados são os locais de memória onde esses dados ficam armazenados para que o processamento aconteça.
Memória Principal
A memória principal é uma forma de memória de acesso aleatório, também chamada de Memória RAM, usada para armazenar dados e instruções que são acessados e processados pela CPU.
Memória ROM
A memória ROM (Read-Only Memory) é um tipo de memória de armazenamento de computador que armazena dados permanentes e que não podem ser modificados ou apagados. A Memória ROM, que também é uma memória principal, é onde são gravados os programas de inicialização de um computador, chamados de Firmwares, e seus principais tipos de aplicação.

Memória Secundária

Memória Secundária é onde as informações são gravadas para uso posterior. São geralmente dispositivos de memória, como HDs, cartões de memória e outros que possam desempenhar esta função.

Padrão SCSI

O Padrão SCSI (Small Computer System Interface) é um conjunto de padrões para conexão e transferência de dados entre dispositivos de computador. Este padrão foi criado para realizar a comunicação entre dispositivos com confiabilidade de transmissão e velocidade rápida.

Padrão IDE/ATA

O Padrão IDE/ATA (Integrated Drive Electronics/AT Attachment) é um padrão de interface para conectar dispositivos de armazenamento de massa, como unidades de disco rígido e unidades de CD/DVD, ao computador. IDE foi o primeiro que integrou ao HD a controladora do dispisitivo, o que representou uma grande inovação, reduzindo os problemas de sicnronismo e tornando seu funcionamento mais rápido.

Padrão SATA

O Padrão SATA (Serial ATA) é um padrão de interface para dispositivos de armazenamento de dados, como discos rígidos e SSDs (Solid State Drives), que oferece taxas de transferência de dados mais rápidas e eficiência energética aprimorada em comparação com padrões anteriores. Sucessor do padrão ATA. Utiliza dois canais, um para enviar e outro para receber dados, reduzindo com isso problemas de sincronização e interfereência.

Dispositivos de memória SSD

Os SSDs (Solid State Drives) são dispositivos de armazenamento de dados que usam memória flash NAND para armazenar dados de forma permanente. Eles são mais rápidos, silenciosos e duráveis do que os discos rígidos tradicionais, mas geralmente têm capacidade de armazenamento menor e custo mais alto por gigabyte. Dispositivos para armazenamento de dados (como os HDs). Formados por chips de memória Flash para realizar o armazenamento de dados. São mais econômicos no consumo de energia.

Dispositivos de entrada e saída

Os dispositivos de entrada e saída (E/S) são componentes fundamentais de um sistema de computador. Eles permitem que os usuários interajam com o computador, enviando dados para dentro e recebendo dados de volta para fora do sistema.

Dispositivo de Entrada

Um dispositivo de entrada é qualquer periférico de computador usado para fornecer dados e controle para o computador. Alguns exemplos comuns incluem teclados, mouse, trackpad, scanner, microfone e câmera.

Dispositivo de Saída

Um dispositivo de saída é qualquer periférico de computador usado para fornecer informações ou resultados do computador para o usuário. Alguns exemplos incluem monitores, impressoras, alto-falantes, fones de ouvido e projetores.
